THE PROJECT'S DESCRIPTION

This project accesses information and reports from the virtual trade world. The virtual trade world has become very popular in recent years. Still, at the same time, a variety of APIs have been created that provide free information (usually with a one-time registration) about the status of currencies, price, history, buying and more.

Additional explanation of the buttons in each coin:
More Info button:
Clicking on this button will open more information about the currency inside the card, showing the currency image and the currency price according to the currencies: USD, Euro, NIS.

Toggle button:
Clicking this button will add the current currency to the list of reports.
Note - Up to 5 coins can be added to the list of reports. If you have selected 5 coins and you want to add a currency, you must choose to remove one of the five coins already selected, that is, display in the reports.

Real-time reports screen
This screen displays a graph depicting the status of the selected coins in the toggle button in real-time.
Each currency selected and marked as ON in the button toggle belongs to the above graph and report.

Description of the graph
1. The graph will contain lines of different colors as the number of coins selected.
2. Each color will be a currency and will be listed in the graph in which color belongs to which currency - by currency code.
3. The information in the graph will be updated every 2 seconds.
4. Each entry to the graph page resets the report, which means that the X-axis representing time, as well as the Y-axis representing the currency price, will be reset.
5. Any addition or download of currency from the report and report will affect the graph accordingly.

Search functionality and filters:
Only the currencies shown in the report can be displayed for removal from the report or view. You can search by currency code only, meaning that BTC will return the information of BITCOIN currency as well as any other The toggle and info more functionality will apply to the results. If you search for BT you will not get results.
